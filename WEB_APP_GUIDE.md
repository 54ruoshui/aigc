# GraphRAG Web应用使用指南

## 🌟 项目简介

计算机网络知识图谱GraphRAG系统是一个基于大语言模型的智能问答系统，结合了Neo4j图数据库和智谱AI API，为用户提供专业的计算机网络知识问答服务。

## 🚀 快速开始

### 1. 环境准备

确保您的系统已安装以下软件：
- Python 3.8+
- Neo4j数据库（版本4.0+）
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置环境变量

复制 `.env.example` 文件为 `.env` 并配置以下参数：

```env
# Neo4j数据库配置
NEO4J_URI=bolt://localhost:7687
NEO4J_USER=neo4j
NEO4J_PASSWORD=your_password
NEO4J_DATABASE=neo4j

# 智谱AI API配置
ZHIPUAI_API_KEY=your_api_key
ZHIPUAI_MODEL=glm-4-flash
ZHIPUAI_BASE_URL=https://open.bigmodel.cn/api/paas/v4

# Web服务器配置
FLASK_HOST=0.0.0.0
FLASK_PORT=5000
FLASK_DEBUG=False
```

### 4. 启动应用

#### 方法一：使用启动脚本（推荐）
```bash
python start_web_app.py
```

#### 方法二：使用原始脚本
```bash
python run.py --mode web
```

#### 方法三：直接运行Flask应用
```bash
python -m web.graph_rag_web
```

启动成功后，浏览器会自动打开 `http://localhost:5000`

## 🎮 使用指南

### 基本操作

1. **提问**
   - 在输入框中输入问题
   - 点击"提问"按钮或按Enter键
   - 支持Ctrl+Enter快捷键提交

2. **语音输入**
   - 点击麦克风按钮
   - 说出您的问题
   - 系统自动转换为文字

3. **查看答案**
   - 答案会显示在下方区域
   - 包含处理时间统计
   - 支持复制和分享功能

### 高级功能

1. **图谱交互**
   - 点击节点查看详细信息
   - 拖拽节点调整位置
   - 滚轮缩放图谱大小

2. **历史管理**
   - 左侧面板显示查询历史
   - 点击历史项可重新提问
   - 支持清空历史记录

3. **导出功能**
   - 导出图谱为PNG图片
   - 保存问答记录
   - 分享给他人

## 🔧 故障排除

### 常见问题

1. **无法连接Neo4j数据库**
   - 检查Neo4j是否正在运行
   - 验证连接参数是否正确
   - 确认数据库用户名和密码

2. **API调用失败**
   - 检查智谱AI API密钥是否有效
   - 验证网络连接是否正常
   - 确认API配额是否充足

3. **语音识别不工作**
   - 确保浏览器支持Web Speech API
   - 检查麦克风权限是否已授权
   - 尝试使用Chrome浏览器

4. **页面显示异常**
   - 清除浏览器缓存
   - 检查JavaScript是否启用
   - 尝试刷新页面

### 日志查看

- 系统初始化状态
- 查询处理过程
