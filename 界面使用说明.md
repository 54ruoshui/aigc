# GraphRAG 界面使用说明

您的项目中有三种不同的界面版本：

## 1. 综合版界面（推荐）
这是最新创建的界面，集成了问答和知识点提取功能，满足您的所有需求。

### 功能特点：
- **智能问答系统** - 支持网络相关问题查询
- **知识点提取** - 从文本中提取知识点和关系
- **知识图谱可视化** - 直观展示知识网络
- **选项卡界面** - 便捷切换不同功能
- **示例问题提示** - 提供常见问题示例
- **语音输入** - 支持语音提问
- **查询历史记录** - 保存和管理查询历史
- **文件上传** - 支持从文件提取知识点
- **数据导出** - 导出图谱数据
- **节点搜索** - 快速查找知识节点

### 启动方式：
1. **推荐方式：使用批处理文件**
   ```
   双击运行：启动综合版界面.bat
   ```

2. **方式二：使用Python脚本**
   ```
   python start_integrated_app.py
   ```

## 2. 基础版界面（问答功能）
这是您之前使用的界面，主要功能是知识图谱问答。

### 功能特点：
- 智能问答系统
- 知识图谱可视化
- 示例问题提示
- 语音输入
- 查询历史记录
- 数据导出功能

### 启动方式：
1. **推荐方式：使用简单版批处理文件**
   ```
   双击运行：启动基础版界面-简单版.bat
   ```

2. **方式二：使用批处理文件**
   ```
   双击运行：启动基础版界面.bat
   ```

3. **方式三：使用Python脚本**
   ```
   python start_basic_web_app.py
   ```

4. **方式四：使用简单版Python脚本**
   ```
   python start_simple_basic_app.py
   ```

5. **方式五：使用原始脚本**
   ```
   python run.py --mode web
   ```

## 3. 增强版界面（知识点提取）
这是当前默认的界面，专注于知识点提取和验证。

### 功能特点：
- 知识点提取
- 技能验证系统
- 反馈学习
- 验证统计
- 文件上传功能

### 启动方式：
1. **方式一：使用批处理文件**
   ```
   双击运行：快速启动Web应用.bat
   ```

2. **方式二：使用Python脚本**
   ```
   python run_enhanced_web_app.py
   ```

3. **方式三：使用原始脚本**
   ```
   python start_web_app.py
   ```

## 如何切换界面

如果您想要从增强版界面切换回基础版界面，最简单的方法是：

1. 停止当前运行的应用
2. 双击运行 `启动基础版界面.bat`
3. 等待应用启动完成，然后访问 http://localhost:5000

## 注意事项

- 两个界面使用相同的后端API，只是前端展示不同
- 基础版界面更适合问答场景
- 增强版界面更适合知识提取和管理场景
- 您可以根据需要随时切换使用不同的界面

## 文件说明

- `templates/index_final.html` - 基础版界面模板
- `templates/enhanced_index.html` - 增强版界面模板
- `templates/index.html` - 重定向页面（自动跳转到增强版）
- `start_basic_web_app.py` - 基础版启动脚本
- `run_enhanced_web_app.py` - 增强版启动脚本
- `启动基础版界面.bat` - 基础版批处理启动文件
- `快速启动Web应用.bat` - 增强版批处理启动文件
